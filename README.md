# 🚗 BMW Global Sales Performance Dashboard

![Dashboard Preview](Dashboard%20Preview.png)

## 📌 Project Overview

This project presents an interactive Tableau dashboard that analyzes BMW's global sales performance. The dashboard enables users to explore revenue trends, product performance, country-wise sales, engine type distribution, and the relationship between marketing spend and sales through interactive filters and visualizations.

---

## 🎯 Business Objective

The objective of this dashboard is to provide business stakeholders with actionable insights into BMW's global sales performance by answering questions such as:

- Which countries generate the highest sales?
- Which vehicle segment contributes the most revenue?
- How has revenue changed over the years?
- Which engine types are most popular?
- Does higher marketing spend lead to higher sales?

## 📌 Key Insights

- SUV segment generated the highest overall revenue.
- Revenue remained relatively stable across multiple years.
- Marketing spend shows a positive correlation with units sold.
- BMW sales are distributed across multiple international markets.
- Engine type sales are fairly balanced across categories.
